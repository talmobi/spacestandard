# spacestandard

<a href="https://github.com/standard/standard">standard</a> with spaces

# sample

```js
var list = [{a:'foo'},{b:'bar'}]

get(function () {
  return ('cake')
})
```

=>

```js
var list = [ { a: 'foo' }, { b: 'bar' } ]

get( function () {
  return ( 'cake' )
} )
```

# Install

locally ( project specific, for use with npm scripts )
```bash
npm install spacestandard
```

globally
```bash
npm install -g spacestandard
```
